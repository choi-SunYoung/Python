1. **리스트(list)**

   ```python
   x = []
   x = list()      #두 가지 표기법이 있다.
   ```

   리스트는 element를 그룹핑하기 위해 쓰는것이다. 리스트를 사용하면 단순히 각각의 값을 변수에 넣는 것 보다 활용도가 높아진다. 그리고 리스트를 활용하는 방법을 잘 정리해둔 사이트가 있다. https://docs.python.org/3/library/stdtypes.html#list

   ```python
   days=["Mon", "Tue", "Wed", "Thur", "Fri"]
   weekend=["Sat", "Sun"]
   print(days)
   print(weekend)
   print(days + weekend)
   days.append("Sat")
   print(days)
   days.reverse()
   print(days)
   ```
